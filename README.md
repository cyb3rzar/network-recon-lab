# network-recon-lab
Author: Juan Medina

## Description
This project demonstrates basic and intermediate network reconnaissance techniques in a controlled and legal lab environment.

## Objective
Simulate how an attacker enumerates hosts, open ports, and services inside an isolated lab.

## Tools Used
- Nmap
- Netcat
- Linux Bash

## Environment
- Attacker machine: Kali Linux
- Target machine: Metasploitable2 (local VM)

## Disclaimer
All tests were performed in a local isolated laboratory for educational purposes only.
